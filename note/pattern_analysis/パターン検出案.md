## **テクニカル分析 資料**

この資料は、ご提示いただいた画像に基づき、「ローソク足分析」「チャートパターン分析」「ライン分析」の3つの主要なテクニカル分析手法をまとめたものです。

### **1. ローソク足分析**

ローソク足一本一本の形や組み合わせから、市場参加者の心理を読み解き、今後の値動きを予測する手法です。

#### **1.1. トレンド転換を示唆するローソク足**

相場の流れが変わる可能性を示すサインです。

| 名称 | 特徴 | 売買シグナル |
| :--- | :--- | :--- |
| **つつみ足** | 前の足の実体を完全に包み込む長い足が出現。 | **陽のつつみ足**：底値圏で出現すると上昇転換の可能性（買いサイン）。<br>**陰のつつみ足**：天井圏で出現すると下落転換の可能性（売りサイン）。 |
| **はらみ足** | 前の足の実体の中に収まる短い足が出現。 | **陽のはらみ足**：底値圏で出現すると上昇転換の可能性（買いサイン）。<br>**陰のはらみ足**：天井圏で出現すると下落転換の可能性（売りサイン）。 |
| **たすき線** | 窓を開けて寄り付いた後、前の足の始値付近まで戻して引ける。 | **上昇たすき**：上昇トレンド中に出現すると、一時的な押し目となり、再度上昇する可能性（買いサイン）。<br>**下落たすき**：下落トレンド中に出現すると、一時的な戻りとなり、再度下落する可能性（売りサイン）。 |
| **かぶせ線** | 天井圏で、前の陽線の終値より高く寄り付き、実体の半分以下まで深く陰線でかぶさる。 | 天井圏での下落転換を示唆（強い売りサイン）。 |
| **上放れたすき** | 底値圏で前の陰線から窓を開けて陽線が出現するが、上ヒゲが長く実体が短い。 | 底値圏での上昇転換を示唆（買いサイン）。 |

#### **1.2. トレンド継続を示唆するローソク足**

現在の相場の勢いがまだ続くことを示すサインです。

| 名称 | 特徴 | 売買シグナル |
| :--- | :--- | :--- |
| **赤三兵 (あかさんぺい)** | 底値圏で陽線が3本連続で出現。終値が前の足より高く更新されていく。 | 強い上昇トレンドの開始を示唆（買いサイン）。 |
| **切り込み線** | 下落トレンド中に、前の陰線の終値より安く寄り付き、実体の半分以上を陽線で切り返す。 | 底値圏での上昇転換を示唆（買いサイン）。 |
| **かいの連続** | 上昇トレンド中に複数の陰線が出現した後、それらを打ち消す大陽線が出現。 | 上昇トレンドの継続を示唆（買いサイン）。 |
| **三手打ち (さんてうち)** | 上昇トレンド中に調整の陰線を挟み、再度上昇するパターン。 | 上昇トレンド継続中の押し目買いのチャンス（買いサイン）。 |
| **大陽線/大陰線引け坊主** | 始値から終値まで一方向に大きく動き、ヒゲがないか非常に短い。 | **大陽線**：非常に強い買いの勢い。<br>**大陰線**：非常に強い売りの勢い。 |
| **陽/陰の寄り付き坊主** | 始値が最安値（陽線）または最高値（陰線）となる形。 | **陽の寄り付き坊主**：強い上昇の勢い。<br>**陰の寄り付き坊主**：強い下落の勢い。 |

### **2. チャートパターン分析**

複数のローソク足が形成する特定の形状から、相場の天井や底、トレンドの転換や継続を判断する手法です。

#### **2.1. 反転型チャートパターン**

トレンドの終わりと新しいトレンドの始まりを示唆します。

| 名称 | 形状と特徴 |
| :--- | :--- |
| **ダブルトップ/ボトム** | **トップ**：同じくらいの高値を2回つけ、M字のような形を形成。ネックラインを下抜けると下落サイン。<br>**ボトム**：同じくらいの安値を2回つけ、W字のような形を形成。ネックラインを上抜けると上昇サイン。 |
| **トリプルトップ/ボトム** | **トップ**：同じくらいの高値を3回つける。ダブルトップより強い天井形成のサイン。<br>**ボトム**：同じくらいの安値を3回つける。ダブルボトムより強い底値形成のサイン。 |
| **三尊天井 (ヘッドアンドショルダー)** | 中央の山が最も高い3つの山を形成。人間の頭と両肩のように見える。ネックラインを下抜けると強い下落サイン。 |
| **逆三尊 (インバーテッドH&S)** | 三尊天井を逆にした形で、底値圏で出現。ネックラインを上抜けると強い上昇サイン。 |
| **ソーサートップ/ボトム** | **トップ**：緩やかな弧を描きながら天井を形成。下落は緩やかに始まる。<br>**ボトム**：お皿の底のような緩やかなカーブを描いて底値を形成。上昇転換を示唆。 |

#### **2.2. 保ち合い（継続）型チャートパターン**

トレンドの途中で発生する一時的な踊り場（調整期間）を示し、その後は元のトレンド方向に動き出すことが多いパターンです。

| 名称 | 形状と特徴 |
| :--- | :--- |
| **フラッグ** | **上昇フラッグ**：上昇トレンド中に現れる、旗のような少し下向きの平行四辺形。上辺をブレイクすると再び上昇する可能性が高い。<br>**下降フラッグ**：下落トレンド中に現れる、旗のような少し上向きの平行四辺形。下辺をブレイクすると再び下落する可能性が高い。 |
| **ウェッジ** | **上昇ウェッジ**：先端が徐々に狭まる上向きの三角形。天井圏で出現すると下落転換、下落トレンドの途中で出現すると下落継続のサイン。<br>**下降ウェッジ**：先端が徐々に狭まる下向きの三角形。底値圏で出現すると上昇転換、上昇トレンドの途中で出現すると上昇継続のサイン。 |

### **3. ライン分析**

チャート上に補助線を引くことで、相場の流れや抵抗・支持帯を視覚的に把握する手法です。

#### **3.1. 水平線**

特定の価格帯に引く水平なラインです。

* **レジスタンスライン（上値抵抗線）**: 何度も価格の上昇を止められている高値を結んだ線。このラインを上抜けると（ブレイクアウト）、強い上昇が見込まれます。
* **サポートライン（下値支持線）**: 何度も価格の下落を支えている安値を結んだ線。このラインを下抜けると（ブレイクダウン）、強い下落が見込まれます。
* **ロールリバーサル（レジサポ転換）**: レジスタンスラインを上抜けた後、そのラインが今度はサポートラインとして機能すること。逆も同様で、サポートラインがレジスタンスラインに変わることもあります。信頼性の高いエントリーポイントになり得ます。

#### **3.2. チャネルライン**

トレンドラインと平行に引いた線のことで、価格が動きやすい範囲（チャネル）を示します。

* **上昇チャネル**: 安値を結んだトレンドライン（サポート）と、それと平行に高値を結んだチャネルライン（レジスタンス）の間で価格が推移します。
* **下降チャネル**: 高値を結んだトレンドライン（レジスタンス）と、それと平行に安値を結んだチャネルライン（サポート）の間で価格が推移します。

#### **3.3. テクニカルインジケーター**

価格データなどを基に計算された指標で、トレンドの強さや売買のタイミングを判断するのに役立ちます。

* **RSI (相対力指数)**
    * 買われすぎか売られすぎかを示すオシレーター系指標。
    * 一般的に**70%以上で「買われすぎ」**、**30%以下で「売られすぎ」**と判断され、逆張りの目安として使われます。
* **ボリンジャーバンド**
    * 移動平均線とその上下に統計学的に計算された価格の変動幅（標準偏差、σ）を示すライン。
    * 価格が**±2σのラインに達すると反発しやすい**という逆張りの見方や、**±1σのラインに沿って動いている間はトレンド継続**（バンドウォーク）という順張りの見方があります。
* **移動平均線 (MA)**
    * 一定期間の価格の平均値を結んだ線で、トレンドの方向性を示します。
    * **ゴールデンクロス**: 短期線が長期線を下から上に突き抜ける現象。強い**買いサイン**。
    * **デッドクロス**: 短期線が長期線を上から下に突き抜ける現象。強い**売りサイン**。
